HOW TO PLAY SUDOKU

To successfully complete a Sudoku puzzle, fill all the empty cells so that every row, column, and square contains all numbers from 1-9. A number can only appear once in each row, column, and block.

HOW TO USE SUDOKU SOLVER

The purpose of SUDOKU SOLVER is to provide a solution for any Sudoku puzzle. Begin by entering an unsolved Sudoku puzzle. Keep in mind that the input has to be a number between 1 and 9 (inclusively). Cells will turn red if the input breaks any constraints or Sudoku rules.

You can either receive hints using the "Show Hint" button or display the full solution using the "Show Solution" button below. If you clicked the "Show Hint" button, select any empty cell to reveal its number.
